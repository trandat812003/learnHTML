HTML introduction

    What is HTML?

  • HTML stand for Hyper Text Markup Language
  • HTML is the standard markup language for create Web page
  • HTML describe the structure of a Web page
  • HTML consist of a series of element
  • HTML element tell the browsers how to display the content

    • What is HTML element?

  • An HTML element id define by a start tag, content, and end tag
