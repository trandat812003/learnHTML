Unordered list

  1. Basic

    An unordered list start with the <ul> tag.
    Item list start <li> tag.
    By default, the list item will be mark bullet
    Syntax:
    <ul>
      <li>text</li>
    </ul>

  2. Choose list item mark

    Syntax:
    <ul style="list-style-type:value">
    </ul>
    Value Description
    disc Set the list item mark to a bullet(default)
    circle set the list item mark to a circle
    square set the list item mark to a square

Ordered list

  1. Basic

    An unordered list start with the <ul> tag.
    Item list start <li> tag.
    By default, the list item will be mark bullet
    Syntax:
    <ul>
      <li>text</li>
    </ul>

  2. The style attribute

    Syntax:
    <ol type="value">
    </ol>
    Type Describes
    type="1" the list item will be number with number (default)
    type="A" the list item will be number with uppercase
    type="a" the list item will be number with lowercase
    type="I" the list item will be number with uppercase roman number
    type="i" the list item will be number with lowercase roman number

Description list

HTML also support description list.
Description list is a list of term, with description of each term
The <dl> tag define description list, the <dt> tag define the term(name), and <dd> tag define describes each term.
Syntax:
<dl>
  <dt>text</dt>
  <dd>text</dd> (x n)
  <dt>text</dt>
  <dd>text</dd> (x n)
</dl>
