LEARN HTML

  1. RGB value

    -syntax:style="property:rgb(valueRED, valueGREEN, valueBLUE);" .

    - each parameter valueRED, valueGREEN, valueBLUE define intensity of the color with a value between 0 to 255.

    RGBA value

    -syntax: style="property:rgba(valueRED, valueGREEN, valueBLUE, alpha);"

    - alpha define intensity of between 0,0(full transparent) and 1.0(not transparent) at all.

  2. HEX value

    - syntax: <tagname style = "property:#rrggbb;">

    - rr, gg, bb, are hexadecimal value between 00 and ff(same decimal 0-255)

  3. HSL value

    - syntax: <tagname style = "property:hsl(hue, saturation, lightness);(hsl(1,100%,100%),...)">

    - hue is degree on the color wheel from 0 to 360 (0 red, 120 green, 240 blue);
    - saturation is a percentage value, 0% is gray, and 100% is full color;
    - lightness is a percentage value, 0% is black, 100% is white;

    HSLA value

    - syntax: <tagname style = "property:HSLA(hue, saturation, lightness, alpha); (hsl(1,50%,50%,0.5),...)">

    - alpha parameters is a number between 0.0(full transparent) and 1.0(not transparent) at all;
