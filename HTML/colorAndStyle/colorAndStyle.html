HTML color and style

The HTML style attribute use add style to an element.
HTML color are specify and predefine color name.

Attribute Syntax Content
style <tagname style = "property:value;"> Add style ta an element.
background-color <tagname style = "background-color:value(blue, red, black);"> Define background color for an HTML element.
color <tagname style = "color:value(blue, red, black,...);"> Define text color for an HTML element.
font-size <tagname style = "font-size:value(200%, 100%,....);"> Define text size for a HTML element
font-family <tagname style = "font-family:value(time new roman, ....);"> Define the font for a HTML element
text-align <tagname style = "text-align:value(center, ....);"> Define the horizontal text alignment for an HTML element
border...solid <tagname style = "border:value(2px,...) solid value;"> Define border and color of border element
