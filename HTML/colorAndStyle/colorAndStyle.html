LEARN HTML

HTML style attribute:

<tagname style = "property:value;">

HTML background-color attribute:

<tagname style = "background-color:value(blue, red, black);"> -> define background color for an HTML element

HTML color attribute:

<tagname style = "color:value(blue, red, black,...);"> -> define text color for an HTML element

HTML font-size attribute:

<tagname style = "font-size:value(200%, 100%,....);"> -> define text size for a HTML element

HTML font-family attribute:

<tagname style = "font-family:value(time new roman, ....);">-> define the font for a HTML element

HTML text-align attribute:

<tagname style = "text-align:value(center, ....);">-> define the horizontal text alignment for an HTML element

HTML border...solid attribute:

<tagname style = "border:value(2px,...) solid value;"> -> define border and color of border element
