HTML block and inline element

Every HTML element has a default display value, depend on what type of element it is.
They are two display: block and inline.

Block-level element

A block-level element alway start on new line.
A block-level element alway take up the full with available (stretches out to left an right as far as it can)
A block-level element has a top and bottom margin,whereas an inline element doesn't
