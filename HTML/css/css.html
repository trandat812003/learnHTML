What is CSS?

  - CSS stand for Cascading Style Sheets.
  - can control layout of multiple web pages.

Using CSS

  - CSS can be add to HTML document in 3 way:

  • Inline: using style attribute inside HTML element

  • Internal: using <style> element in the head section

  • External: using <link> element to link to an external CSS file

Inline:

   - syntax:<tagname style="property:value">
   - is using apply a unique style to a single HTML element.
   - use style attribute HTML element.

Internal:

   - syntax:
<!DOCTYPE html>
<html>
     <head>
         <style>
         ... code...
         </style>
     </head>
</html>
    - using to define a style for a single HTML page.
   - define in the <head> section, within a <style>

External:

   - syntax: <link rel="stylesheet" href="path">
   - use define style the style for many HTML pages.
   - add link to it in the <head> section of each HTML pages.
