HTML iframe

An HTML iframe is use to display a web within a web page.

HTML iframe syntax

The HTMl <iframe> tag specify an inline frame.
An inline frame is use to embed another document within the current HTML document.
Syntax: <iframe src="url" title="description"></iframe>
Tip: It's good practice to alway include title attribute for the <iframe> This is screen reader to read out what content of the iframe.

Iframe set width and height

Use width and height attribute specify size frame.
The height and width are specify pixel by default.
Syntax: <iframe src="url" title="description" width="number" height="number"></iframe> Or use style attribute and use CSS height and width property.
Syntax: <iframe src="url" title="description" style="width:number;height:number;"><iframe>

Iframe remove border

By default, iframe has border around it.
To remove the border, add the style attribute and CSS border property set none.
With CSS, you can change style, size, color of iframe border.
