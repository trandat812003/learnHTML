HTML link

HTML link are hyper link.
A link doesn't have to be text. link can be img or any other html element.

HTML link syntax

Syntax: <a href="url" target = "text" > linktext </a>
The most important attribute of the <a> is the <href> attribute; Linktext is the part visible to the reader.
The target attribute specify where to open the link document.

  • _self: default. open the window/tab as it is click
  • _blank: open the new window/tab
  • _parent: open in the parent fame
  • _top: open in the full body of the window

Using an img as a link

To use an img as a link, just push <img> tag inside the <a> tag.
<a href='URL'>
<img src="URL" alt="alt">
</a>

Link title

The title specify information about an element.
The information is most show when the mouse move over element.
Syntax: <a href="URL" title="tooltip">content</a>

CSS link

  1. a:link -> text link
  2. a:visited -> visited link
  3. a:active -> active link
  4. a:hover -> mouse over a link

Link bookmark

Bookmark can useful if a web page is long
To create to bookmark, then add link to it
When click, web will scroll down or scroll up to the location with bookmark
Using id attribute create bookmark.
Syntax: <tagname id="value">
Use the href attribute to link the bookmark.
Syntax: <tagname href="#value">
