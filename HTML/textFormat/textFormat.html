LEARN HTML

  1. b and strong: bold text

    syntax: <b>content</b> and <strong>content</strong>

  2. i and em: italic text

    syntax: <i>content</i> and <em>content</em>

  3. small:define small text

    syntax:<small>content</small>

  4. mark: define text that should be mark or highlighted

    syntax:<mark>content</mark>

  5. del:define text that has been deleted from a document

    syntax:<del>content</del>

  6. ins:define a text that has been insert into a document

    syntax:<ins>content</ins>

  7. sub: define subscript text

    syntax:<sub>content</sub>

  8. sup: define superscript text

    syntax:<sup>content</sup>
