HTML text format

HTML contain server element for define text with special meaning.

  1. <b> and <strong>

    Bold text
    Syntax: <b>content</b> and <strong>content</strong>

  2. <i> and <em>

    Italic text
    Syntax: <i>content</i> and <em>content</em>

  3. <small>

    Define small text
    Syntax:<small>content</small>

  4. <mark>

    Define text that should be mark or highlighted
    Syntax:<mark>content</mark>

  5. <del>

    Define text that has been deleted from a document
    Syntax:<del>content</del>

  6. <ins>

    Define a text that has been insert into a document
    Syntax:<ins>content</ins>

  7. <sub>

    Define subscript text
    Syntax:<sub>content</sub>

  8. <sup>

    Define superscript text
    Syntax:<sup>content</sup>
