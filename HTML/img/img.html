HTML img syntax

<!DOCTYPE html>
<html>
  <body>
    <img src="URL" alt="some-text">
  </body>
<html>

The alt attribute

the alt attribute specify an alternate text for an img, if the img can't be display
if browser can't find img, it will display the alt text.

Img size

you can use the style attribute to specify the width and height.
syntax: <img src="URL" alt="some-text" style="width:value;height: value">

Using an img as a link

you can use an img as a link, simply nest the <img> tag inside the <a>
syntax:
    <a href="url-link">
        <img src="url-img" alt="text">
    </a>

Img floating

the CSS float property to let the img float
the img can float to the right or the left of a text
syntax:
    <img src="url" alt="text" style="float: right"> : the img will float to the right of the text
    <img src="url" alt="text" style="float: left"> : the img will float to the left of the text

  1. The Img

    The img insert <img> tag. the only difference from other img is that must usemap attribute
    The usemap value start with a hash tag # follow by the name of img map, and relationship between the img and the img map.
    syntax: <img src="" alt="" usemap="#workmap">

  2. Create img map

    Add a map element
    The map element use create img map, and link to the img by use require name attribute
    syntax: <map name="#workmap">

  3. The area:

    a clickable areas define using an <area> element

      Shape

    • rect - define a rectangular region
    • circle - define a circle region
    • poly - define a polygonal region
    • default - define the entire region

Background img on HTML element

add background img on a HTML element, use HTML style attribute and CSS background-img property
syntax: <tagname style="background-img: url('url') "

background img on page

specify the background img on the <body> element
syntax: <body style="background-img: url('url') "

background repeat

if background ing small element, img will repeat itself, horizon and vertical, until it reacher the end of the element
to avoid the background from repeat itself, set background-repeat property no-repeat
syntax:
<style>
   element {
    background-repeat: no-repeat;
  }
</style>

background cover

if you want background img entire, can set background-size property to cover
also, to make sure the entire element no the stretching( the img will keep original proportion), set the background-attachment property to fixed
<style>
   element {
    background-repeat: no-repeat;
    background-attachment: fixed;
    background-size: cover;
  }
</style>

background stretch

if you want the background img to stretch to fit the entire element, you can set the background-size property to 100% 100%
<style>
   element {
    background-repeat: no-repeat;
    background-attachment: fixed;
    background-size: 100% 100%;
  }
</style>
