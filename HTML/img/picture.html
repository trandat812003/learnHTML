HTML picture

The HTML <picture> element allow you display different picture for different device or screen size.

HTML <picture> element

The HTML picture element give web developer more flexibility specify img resource.
The picture element contain one or more source element, each referring to different img the srcset attribute -> the browser can choose the img best fit the current view and/or device.
Each source element has a media attribute the define when img is the most suitable.
Syntax:
<picture>
    <source media="(property: value)" srcset="url">
    <source media="(property: value)" srcset="url">
    <source media="(property: value)" srcset="url">
    <img src="url" alt="text">
</picture>

When to use the Picture element

  1. Bandwidth

    If you have a small device or screen, it's not necessary to load large img. The browser will use the first <source> element with matching attribute value, and ignore any of follow element

  2. Format support

    Some browser or device not support img format. By using the <picture> element, you can add img of all format, and the browser will the first format it recognizes, ignore any of the follow element.
