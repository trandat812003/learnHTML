HTML class

The HTML class attribute is use to specify a class for an HTML element.
Multiple HTML element can share same class.

Using the class attribute

The class attribute often used to point to a class name in a CSS. It can also be use JS to access am manipulate element with specify class name.

  • The class attribute can be use on any HTML element.
  • The class name is case sensitive.

The syntax for class

To create class; write a period (.) character, follow class name. Then, define CSS property within curly braces {}

Multiple class

HTML element can belong to more than one class.
To define multiple class, separate the class name with a space. The element will be style according to all the class specify.
e.g <div class="city main">.
the <div> element belong to both city class, main class and will get CSS style from both of the class.
