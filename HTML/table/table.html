Define HTML table

the <table> tag define HTML table

  • <tr> tag define table row
  • <th> tag define table header
  • <td> tag define table data/cell

they can contain all sort of HTML element, text, img, list, other table and more.
syntax:

<table>
    <tr>
        <th> text-header <th>
        <th> text-header <th>
        <th> text-header <th>
        ....
    </tr>
    <tr>
        <tr> text-data <tr>
        <tr> text-data <tr>
        <tr> text-data <tr>
        ....
    </tr>
</table>

By default, text in <th> element are bold and center
By default, text in <td> element are regular and left-align
