Define HTML table

the <table> tag define HTML table

  • <tr> tag define table row
  • <th> tag define table header
  • <td> tag define table data/cell

they can contain all sort of HTML element, text, img, list, other table and more.
syntax:

<table>
    <tr>
        <th> text-header <th>
        <th> text-header <th>
        <th> text-header <th>
        ....
    </tr>
    <tr>
        <tr> text-data <tr>
        <tr> text-data <tr>
        <tr> text-data <tr>
        ....
    </tr>
</table>

By default, text in <th> element are bold and center
By default, text in <td> element are regular and left-align

Add border

add a border to a table, use CSS border property.
syntax:
table, th, tr {
   border: value solid value;(border 1px solid red)
}
remember to define border the table and the table cell

Add cell padding

cell padding specify space between cell content and its border.
if you not specify padding, the table cell will display without padding.
to set padding, the use CSS padding attribute.
syntax:
th, tr {
   padding: value;(padding: 15px)
}

left-align heading

By default, table heading are bole and center.
to left-align the table heading, use CSS text-align property.
syntax:
th {
  text-align: left;
}

Add border space

Border space specify the space between cell.
to set border space, use CSS border-space property.
syntax: table{
  border-space: value;(border-space: 5px;)
}

cell that span many column

to make a cell span more than once column, use colspan property.
syntax:
<table>
  <tr>
    <th>text</th>
    <th colspan="number">text</th>
  </tr>
  <tr>
    <td>text</td>
    <td>text</td>
    <td>text</td>
  <tr>
</table>

cell that span many row

to make a cell span more that once row, use colspan property.
Ex:
<table>
  <tr>
    <th>text</th>
    <td>text</td>
  </tr>
  <tr>
    <th rowspan="3">text</th>
    <td>text</td>
  </tr>
  <tr>
    <td>text</td>
  </tr>
  <tr>
    <td>text</td>
  </tr>
</table>

Add caption

to a add caption to a table, use the <caption> tag.
syntax:
<table>
  <caption>text</caption>
  <tr>
    <   </tr>
</table>

A specify style for one table

To define a specify style for one particular table, add an id attribute to the table.
Ex:

</table>
