HTML id

The HTML id attribute is use to specify a unique id for HTML element.
You cannot have more than one element with the same id in HTML document.

Using the id attribute

The id attribute specify a unique for HTML element. the value of the id attribute must be within HTML document.
The id attribute is use to point to a specific style declaration in a stylesheet. It's also use by JS to access and manipulate the element with specific id.
The syntax for id is: write hash character (#), follow by an id name, define CSS property within curly braces {}.

HTML bookmark with id and link

HTML bookmark are use to allow reader jump specify part of a webpage.
Bookmark can be useful if your page very long.
To use bookmark, you create id and add link to it.
When the link click, the page scroll to the location with the bookmark
